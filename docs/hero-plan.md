# 메인페이지 개편

## 목표

### 메인페이지

- 3D 캔버스 부분을 클릭 가능으로 만들어 클릭하면 별도의 페이지(3d explore 페이지)로 이동
- 메인 페이지의 캔버스 부분에는 대표되는(나를 나타내는) 3D 모델을 배치
- 해당 3d 모델을 스핀하는 애니메이션을 적용

### 3D explore 페이지(메인페이지에서 3d 캔버스를 클릭하면 이동하는 페이지)

> 3D 공간에서 프로젝트를 탐색하는 페이지

- 공간 배치
  - 가운데 나를 대표하는 3d 모델 배치
  - 주변으로 내 프로젝트들을 3d 모델로 배치
  - 각 프로젝트 3d 모델은 클릭하면 해당 프로젝트의 상세 페이지로 이동 (모달로 띄워도 괜찮을듯?)
